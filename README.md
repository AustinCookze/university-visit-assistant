# 高校拜访智能助手

## 简介

专为**科大讯飞高教业务**销售人员设计的智能工具，帮助您在拜访高校领导前完成情报搜集、领导画像分析与拜访策略制定。

## 功能特点

### 1. 智能情报搜集
- 学校基本信息（层次、类型、地理位置）
- 学科建设情况
- 信息化现状与智慧校园建设
- 近期新闻动态
- **领导履历（附参考来源链接）**

### 2. 领导画像分析
- 管理风格识别（标注信息来源）
- 决策特点分析（标注信息来源）
- 关注点预测（标注信息来源）
- 沟通偏好建议（标注可信度）
- **重要声明：严禁臆造领导个人信息**

### 3. 拜访策略生成
- 开场白建议
- 话题牵引路径
- 价值呈现方案
- 可能问题与应对预案
- 下一步行动建议

### 4. 模拟演练
- AI扮演领导进行对话
- 实时练习应答
- 提升临场应变能力

### 5. 完整省份支持
- 支持全部34个中国省级行政区选择
- 包括23个省、5个自治区、4个直辖市、2个特别行政区

## 大模型API配置（可选但推荐）

应用支持配置大模型API Key，配置后将使用真实AI能力生成内容，效果更精准。

### 支持的API提供商

| 提供商 | Endpoint | 默认模型 |
|--------|----------|----------|
| Kimi | https://api.moonshot.cn/v1/chat/completions | kimi-latest |
| OpenAI | https://api.openai.com/v1/chat/completions | gpt-4 |
| 自定义 | 自定义地址 | 自定义 |

### 配置步骤

1. 点击右上角「设置」按钮
2. 选择API提供商或选择「自定义」
3. 填写API Key（必填）
4. 点击「测试连接」验证配置
5. 点击「保存设置」

**注意：** API Key仅存储在浏览器本地，不会上传到任何服务器。

### 获取API Key

- **Kimi**: 访问 https://platform.moonshot.cn 注册并创建API Key
- **OpenAI**: 访问 https://platform.openai.com 获取API Key

## 使用方法

### 方式1：直接打开（最简单）
直接用浏览器打开 `index.html` 文件即可使用。

### 方式2：本地服务器（推荐）
```bash
# 进入应用目录
cd university-visit-assistant

# 启动本地服务器
python3 -m http.server 8080

# 浏览器访问
http://localhost:8080
```

### 方式3：部署到服务器
将文件上传到任意Web服务器或静态托管服务（如GitHub Pages、Vercel、Netlify等）。

## 使用流程

1. **录入信息** - 填写学校、领导、拜访目标等基础信息
2. **情报搜集** - 系统自动生成学校情报（配置API后更精准）
3. **领导画像** - 分析领导风格与关注点
4. **策略生成** - 生成完整的拜访策略方案
5. **模拟演练** - AI扮演领导进行对话练习

## 数据存储

- 所有数据存储在浏览器本地（LocalStorage）
- API Key仅本地保存，不会上传
- 历史记录最多保存50条

## 技术说明

- 纯前端实现，无需后端服务
- 使用 Tailwind CSS 构建界面
- 使用 Font Awesome 图标
- 支持响应式设计，可在手机/平板使用
